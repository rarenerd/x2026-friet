# Liedtekst-schrijfgids вҖ” Friet met Desire

Hoe schrijf je tekst die past op deze melodie? Deze gids beschrijft
het **ritme**, de **klemtonen** en de **melodische lijn** per frase.
Schrijf tekst die het ritme volgt вҖ” de melodie doet de rest.

## Toonsoort en bereik

- **Toonsoort:** D mineur
- **Bereik:** la (A4) tot FA (F5) вҖ” 8 halve tonen, borststem
- **Tempo:** 175 BPM (happy hardcore) / 130 BPM (origineel)
- **Maatsoort:** 4/4

## Structuur

Het lied bestaat uit **vier secties** die herhalen:

```
INTRO (instrumentaal, 4 maten)
COUPLET (8 maten: 4Г— frase A + 4Г— frase B)
PRE-REFREIN (8 maten: 4Г— frase C + 4Г— frase D)
REFREIN (8 maten: 4Г— frase E + 3Г— frase F + afsluiting)
NA-NA (8 maten: zelfde als couplet, andere tekst)
[herhaal refrein 2Г— met variatie]
```

## De zes frasen вҖ” ritme-sjablonen

Elk sjabloon toont:
- **Ritme:** welke tel elke lettergreep op valt
- **Accenten:** HOOFDLETTER = sterke klemtoon
- **Noten:** de melodische richting (вҶ— omhoog, вҶҳ omlaag, вҶ’ gelijk)
- **Lettergrepen:** hoeveel lettergrepen passen

## Rijmschema

```
Couplet:   A  (frase A: stijgend)
           B  (frase B: dalend вҖ” rijmt op vorige B)
           A  (herhaling)
           B  (rijmt)
           ...

Refrein:   E  (PIEK вҖ” altijd dezelfde tekst = "Friet met Desire")
           F  (vrij вҖ” kan variГ«ren per herhaling)
           E  (herhaling)
           F  (variatie)
           ...
```

De refrein-E-regels zijn typisch **altijd hetzelfde** (dat maakt het
een refrein). De F-regels mogen variГ«ren.

## Na-na

De na-na sectie gebruikt dezelfde melodie als het couplet (frase A + B)
maar dan met simpele "na-na-na" tekst of een alternatieve tekst.
Perfecte plek voor **greetings**, **credits** of een **ad-lib** tekst.
